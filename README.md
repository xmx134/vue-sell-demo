# vue-sell-cube
本项目是基于 cube-ui 和 Vue 框架构建的一个移动网页版外卖项目
# 运行环境
项目基于Vue-cli3.0环境构建的，根据Vue-cli3.0进行环境适配就可以了。项目css代码使用stylus类型
